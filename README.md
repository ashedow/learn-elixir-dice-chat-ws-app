# DiceRollerChatApp

(Learn Elixir)

MVP Chat with dice roll, writen on Elixir (with cowboy) over WebSockets.

## Usage

Install Dependency

```bash
mix deps.get
```

Comile dependency
```bash
mix compile
```

Run tests:
```bash
mix test
```

Run app, for exampel, like
```bash
iex -S mix
```
or
```bash
mix run --no-halt
```

Visit [`localhost:4000`](http://localhost:4000)
